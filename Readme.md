# merge-attrs

  Merge attributes of b into a

## Installation

  Install with [component(1)](http://component.io):

    $ component install yields/merge-attrs

## API

#### merge(a, b)

  Merge `b`'s attributes with `a`, skipping unspecified, `id`, `class`
  and attributes that `a` already has.
